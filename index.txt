To view your new LeftLane.io landing page:

1. Open a terminal in the leftlane.io directory

2. Start a local server using one of these commands:

   Python 3:
   python3 -m http.server 8080
   
   OR Node.js:
   npx serve .
   
   OR PHP:
   php -S localhost:8080

3. Open your browser and go to: http://localhost:8080

4. You should see the new landing page!
